WannaCry Pop Quiz

The WannaCry ransomware was famously stopped by a security researcher that took over its kill-switch domain.

When the ransomware ran on a victim, it first checked to see if that certain domain was registered.

The domain name looks like this: iuqerfsodp9ifjaposdfjhgosurijfaewrwergwea[.]com

This domain is known as an Indicator of Compromise (IOC) for WannaCry, and it's the most famous IOC of 2017.

DNS resolutions for iuqerfsodp9ifjaposdfjhgosurijfaewrwergwea[.]com should be a major red flag for anyone monitoring the network because it implies that WannaCry ransomware is actively running!

For this pop quiz we don't need to run any malware- we'll just generate DNS lookups for the IOC domain.
