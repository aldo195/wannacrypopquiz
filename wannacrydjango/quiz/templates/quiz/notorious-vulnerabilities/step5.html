What does this mean?

The discovery of Shellshock made it surprisingly easy to take over vulnerable Unix systems, including web servers and network devices. No surprise then that it spawned dozens of botnets in a matter of days.

How many of my hosts are affected?

Check for the following words in your VA report using whatever filter/search capability you have. In most document viewers, CTRL+F will suffice.

  • Shellshock
  • CVE-2014-6271

For each match, you should be able to see the IP address or hostname. Each unique IP or hostname counts as one affected host.

What is a CVE?

The CVE system is a public standard for tracking vulnerabilities, assigning a number to security holes as they are published. It makes it easy to be clear on which vulnerability we're talking about and the year in the number enables you to tell at a glance how old is a particular security hole in your network.
