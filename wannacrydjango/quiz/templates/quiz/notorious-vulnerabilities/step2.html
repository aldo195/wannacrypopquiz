What does this mean?

EternalBlue is the name of a military-grade vulnerability in the Windows operating system that was discovered and weaponized by the NSA. When an unsavory group of hackers known as The Shadow Brokers stole and released this cyberweapon in 2017, it was quickly put to use by cybercriminals. The most famous example of malware using EternalBlue to spread is the WannaCry ransomware.

How do I check?

Search for the following numbers or words in your scan report using whatever filter/search capability you have. In most document viewers, CTRL+F will suffice.

  • 91360 (in Qualys reports)
  • 97737 (in Nessus reports)
  • EternalBlue
  • WannaCry
  • CVE-2017-0144
  • MS17-010

For each match, you should be able to see the IP address or hostname. Each unique IP or hostname counts as one affected host.

Tip for getting host count quickly

Some vulnerability reporting tools enable you to group affected hosts by vulnerability.
